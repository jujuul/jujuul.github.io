我自用的一些vscode插件分享

发布于 2020-08-21 | 4分钟 | 883字数

今天逛思否的时候，发现一个推荐vscode插件的文章，里面推荐了8个插件，竟然有6个是我没用过的，也借这个机会，整理一下自己用的插件

  1. Beautify
    听名字就能猜到功能了，整理代码用的，500W+的下载量足以证明其强大。
  2. Auto Close Tag
    也是个听名字就能猜到功能的，320W+下载量。
  3. Auto Rename Tag
    跟上面的是同一个作者开发的，可以在更改标签名的时候做到更改一边而两边都被更改的效果，340W+。
  4. Bracket Pair Colorizer
    我个人强力推荐，没完没了的函数中看代码真的是一种折磨，有了它就能清晰的看出(){}[]是否闭合，再也不用担心少个)}]了，340W+。
  5. Chinese (Simplified) Language Pack for Visual Studio Code
    简中语言包，600W+足以证明，另外我去看了下繁中版，也有270W+，中文开发者真的多啊。
  6. Code Runner
    可以直接运行其他语言，使Vs Code成为一个IDE?我是之前毕设项目后端使用PHP才安装的这个插件，不过好像没什么用，话说我在Vscode上配置过java和python的运行环境，效果都不是很理想啊，编辑器和IDE还是有很大区别的啊，466W+。
  7. CSS Peek
    今天被安利的插件，鼠标悬停在类名或者ID上能直接看到CSS规则，100W+。
  8. Debugger for Chrome
    就是在vscode中启动chrome控制台，需要一些比较繁琐的配置，我只是安装了，但从没用过，620W+。
  9. indent-rainbow
    可以和前面第四个联用，这个是在缩进的地方用彩色填充，也是起到一个很好的查看代码的效果,100W+。
  10. JavaScript (ES6) code snippets
    今天受的安利之一，可以生成ES6的代码片段，原来VSCODE不可以的吗？我只生成过VUE的代码片段，但是vue不是使用的ES6语法吗？不是很理解，380W+。
  11. Live Server
    好像可以用来做服务器？而且是自动刷新(热插拔)的那种？也是没有使用过，服务器都是自己用nodejs搭的，原来还有这么方便的，67W+。
  12. open in browser
    在浏览器中打开，还可以选择浏览器，非常实用的功能，220W+。
  13. Prettier - Code formatter
    也是一个代码格式的插件，我记得使用时有要求选择alt+thift+f做出选择，我选了Beautify，我原本以为是冲突的，但今天看了很多介绍用它们两个整理格式的，好像并不冲突？有时间去配置一下，790W+。
  14. Quokka.js
    今天的安利之一，直接放图吧76W+。
  15. REST Client
    REST Client 扩展工具允许你发送 HTTP 请求并直接在 VSCode 中查看响应。再也不需要使用外部应用程序向服务器发送 HTTP 请求，还没用，不过看了动图我觉得可以试试100W+。
  16. TabNine
    一个代码补全工具，基于人工智能，自动补全时会给出每个候选项的概率，并且按照概率大小进行排序，此外，会给出候选项的来源及地址，这样更加方便查询和阅读，31W+。
  17. Vetur
    vue开发必装，之前不知道干嘛的，今天看了一下，是对于vue文件中的语法检查、语法高亮还有自动补全，530W+。
  18. ESLint
    JS代码检测工具，配合之前的几个代码检测，写出的代码将无敌，可惜不会写，配置比较繁琐，1000W+。
